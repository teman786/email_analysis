# 📧 Advanced SOC Email Analyzer

Advanced SOC Email Analyzer is a Python-based cybersecurity tool designed for email threat analysis and phishing detection. The tool helps identify suspicious emails by analyzing headers, extracting URLs and IP addresses, detecting phishing keywords, identifying spoofed domains, and tracking mail flow through received headers.

Built with a graphical user interface using Tkinter, the project simulates real-world SOC investigation workflows used by security analysts and incident responders during phishing and email-based threat investigations.

## 🔍 Key Features

* Email Header Analysis
* Phishing URL Detection
* Sender Domain Verification
* Email Spoofing Detection
* IP Address Extraction
* Mail Flow Tracking
* Suspicious Attachment Detection
* Phishing Keyword Identification
* IOC Extraction
* Automated Email Risk Assessment
* GUI-Based Investigation Tool

## 🛠 Technologies Used

* Python
* Tkinter
* Regex
* URL Parsing
* Email Analysis Techniques

# project goal ----------------------------------------------

The goal of this project is to strengthen practical SOC investigation skills, automate phishing analysis tasks, and improve understanding of email-based cyber threats in a controlled lab environment.
